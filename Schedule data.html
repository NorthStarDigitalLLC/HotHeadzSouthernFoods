Hot Headz Setup
First time setup — this takes about 5 minutes and only needs to be done once. Follow these steps to connect your schedule to Google Sheets so all employees see live data.
  1. 1
    Go to sheets.google.com and create a new blank spreadsheet. Name it "Hot Headz Schedule"
  2. 2
    In the spreadsheet, click Extensions → Apps Script
  3. 3
    Delete all existing code in the editor, then paste the code from the file apps-script.js (included in your download)
  4. 4
    Click Save, then click Deploy → New Deployment
  5. 5
    Set type to "Web App", set "Execute as" to Me, set "Who has access" to Anyone. Click Deploy.
  6. 6
    Copy the Web App URL it gives you and paste it below
⚠️ Already set up before? If you have a URL saved, enter it above and click Save & Connect.
